# 🗄 Database Management System Lab  
**Course Code:** 214456  
**Second Year – Information Technology (2019 Course)**  
Savitribai Phule Pune University, Pune  

---

## 📌 Prerequisite
- Data Structures  
- Software Engineering Principles  

---

## 🎯 Course Objectives

1. Understand database design and implementation concepts.  
2. Learn SQL and industry database practices.  
3. Study conceptual, logical, and physical database design.  
4. Learn relational database systems and architectures.  
5. Implement PL/SQL programs (procedures, functions, triggers, cursors).  

---

## 🎓 Course Outcomes

- **CO1:** Install and configure database systems.  
- **CO2:** Analyze ER models.  
- **CO3:** Design database schema.  
- **CO4:** Implement relational database systems.  
- **CO5:** Use SQL (DDL, DML, DCL).  
- **CO6:** Design backend database (Case Study).  

---

# 📑 Suggested List of Laboratory Assignments

---

## 🔹 Group A – Study of Databases *(CO1)*

1. Study MySQL (Features, Scalability, Performance).  
2. Install & Configure MySQL (Client & Server).  
3. Study SQLite (Overview, Uses, Installation).  

---

## 🔹 Group B – MySQL *(CO2, CO3, CO4, CO5)*

1. Design database (Minimum 3 Entities) with ER/EER Diagram.  
2. Implement database using DDL & apply Normalization.  
3. Create Tables with Primary & Foreign Keys.  
   - Alter Table  
   - Drop Table  
4. Perform SQL Queries:  
   - Relational Operators  
   - Boolean Operators & Pattern Matching  
   - Arithmetic & Built-in Functions  
   - Group Functions  
   - Date & Time Functions  
   - Complex Queries & Set Operators  
5. Implement Views & update base table using Views.  

---

## 🔹 Group C – PL/SQL *(CO6)*

1. Stored Procedure & Function.  
2. Database Triggers (Row & Statement Level).  
3. Cursors (All Types).  

---

## 🔹 Group D – Relational Database Design *(CO5, CO6)*

Case Study: Backend Design of any Organization  

- Project Proposal  
- High-Level SRS  
- Team of 3–4 Members  
- Requirements Document:  
  - Goals  
  - User Types  
  - Functionalities  
  - Hardware & Software Requirements  
  - Expected Load & Database Size  
- ER Diagram
